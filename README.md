### 本地运行

以下内容由脚手架工具自动生成，供参考。

#### Prerequisites
```
git clone https://github.com/TongxinWong/dataset-annotation-tool.git
cd dataset-annotation-tool
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```

#### Compiles and minifies for production
```
npm run build
```

#### Lints and fixes files
```
npm run lint
```
